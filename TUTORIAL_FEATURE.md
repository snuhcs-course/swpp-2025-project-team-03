# 튜토리얼 기능 구현 완료

## 📚 개요

VoiceTutor 앱에 처음 사용하는 사용자를 위한 튜토리얼 기능이 추가되었습니다. 사용자 역할(선생님/학생)에 따라 각각 다른 튜토리얼이 표시됩니다.

## ✨ 주요 기능

### 1. 역할별 튜토리얼
- **선생님 튜토리얼**: 반 관리, 과제 생성, 학생 관리, 성과 분석 등
- **학생 튜토리얼**: 대시보드 사용법, 과제 수행, 진도 확인, 선생님과 소통 등

### 2. 자동 표시
- 처음 앱을 사용하는 계정의 경우 자동으로 튜토리얼 표시
- 한 번 완료하면 다시 표시되지 않음
- 설정에서 언제든지 다시 볼 수 있음

### 3. 사용자 친화적 UI
- 단계별 진행 표시
- 아름다운 애니메이션
- 건너뛰기 옵션 제공
- 이전/다음 버튼으로 자유로운 탐색

## 📁 구현된 파일

### 1. 핵심 컴포넌트
- `utils/TutorialPreferences.kt`: 튜토리얼 완료 상태 관리
- `ui/components/TutorialOverlay.kt`: 튜토리얼 UI 컴포넌트
- `data/models/TutorialData.kt`: 튜토리얼 콘텐츠 정의

### 2. 통합된 화면
- `ui/screens/StudentDashboardScreen.kt`: 학생 대시보드에 튜토리얼 통합
- `ui/screens/TeacherDashboardScreen.kt`: 선생님 대시보드에 튜토리얼 통합
- `ui/screens/SettingsScreen.kt`: 설정에 튜토리얼 초기화 옵션 추가

## 🎯 사용 방법

### 처음 사용자
1. 회원가입 후 로그인
2. 역할(선생님/학생)에 맞는 대시보드 진입 시 자동으로 튜토리얼 표시
3. 튜토리얼을 통해 앱 기능 학습
4. "시작하기" 버튼 클릭으로 완료

### 튜토리얼 다시 보기
1. 설정 화면 진입
2. "튜토리얼 다시 보기" 옵션 선택
3. 확인 다이얼로그에서 "초기화" 클릭
4. 다음 로그인 시 튜토리얼이 다시 표시됨

## 🔧 기술 상세

### TutorialPreferences
SharedPreferences를 사용하여 튜토리얼 완료 상태를 로컬에 저장:
- `teacher_tutorial_completed`: 선생님 튜토리얼 완료 여부
- `student_tutorial_completed`: 학생 튜토리얼 완료 여부

### TutorialOverlay
Material 3 디자인을 따르는 풀스크린 오버레이:
- Dialog를 사용하여 화면 전체를 덮음
- AnimatedContent로 부드러운 전환 효과
- 진행 표시기(점)로 현재 단계 표시
- 이전/다음 버튼과 건너뛰기 옵션

### TutorialData
각 역할별로 5단계의 튜토리얼 스텝 정의:
- 제목, 설명, 아이콘, 색상 포함
- Material Icons 사용
- 역할별로 맞춤화된 콘텐츠

## 🎨 디자인 특징

1. **일관성**: VoiceTutor 디자인 시스템 준수
2. **접근성**: 명확한 텍스트와 충분한 대비
3. **애니메이션**: Spring 애니메이션으로 자연스러운 움직임
4. **반응성**: 모든 화면 크기에 대응

## 📋 튜토리얼 내용

### 선생님 튜토리얼
1. 환영합니다 - VoiceTutor 소개
2. 반 관리 - 반 생성 및 학생 등록
3. 과제 생성 - AI 기반 과제 생성
4. 학생 관리 - 개별 학생 추적
5. 성과 분석 - 데이터 기반 인사이트

### 학생 튜토리얼
1. 환영합니다 - VoiceTutor 소개
2. 대시보드 - 과제 및 성적 확인
3. 과제 수행 - 음성/텍스트로 답변
4. 진도 확인 - 학습 진도 추적
5. 선생님과 소통 - 메시지 기능
